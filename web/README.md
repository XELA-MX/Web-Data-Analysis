# web (React + TypeScript)

Dashboard de tendencias del mercado laboral tech. Lo que el reclutador ve y toca.

- **Stack:** React + TypeScript + Vite.
- **Gráficas:** Recharts / visx.
- Tecnologías más demandadas, rangos salariales, distribución geográfica / % remoto,
  buscador con filtros.
- Más adelante (Fase 4.5): onboarding, ajustes y feed personalizado.

> **Estado:** scaffold (Fase 0). La implementación empieza en la **Fase 4**. Ver
> [`documentación/V1/05-roadmap.md`](../documentación/V1/05-roadmap.md).
