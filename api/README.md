# api

API REST: búsqueda/filtros, agregaciones y series temporales sobre `jobs`.

- **Recomendado para el MVP:** FastAPI (Python) por las agregaciones y los docs
  OpenAPI automáticos. Alternativa: Go (`chi`/`gin`) para un backend unificado.
- Más adelante (Fase 4.5) sirve el feed personalizado por usuario autenticado.

> **Estado:** scaffold (Fase 0). La implementación empieza en la **Fase 4**. Ver
> [`documentación/V1/05-roadmap.md`](../documentación/V1/05-roadmap.md).
